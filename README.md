# Livraria Java

Projeto desenvolvido durante a **Formação Java** da [Rocketseat](https://www.rocketseat.com.br/).

## Sobre

Sistema simples de biblioteca via terminal que permite listar livros disponíveis e realizar empréstimos.

## Funcionalidades

- Cadastro de autores e livros
- Listagem de livros disponíveis
- Empréstimo de livros por nome do cliente

## Tecnologias

- Java
- Orientação a Objetos (POO)
